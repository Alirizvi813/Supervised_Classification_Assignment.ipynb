# Supervised_Classification_Assignment.ipynb
Supervised Classification Assignment
Project Overview
This project builds supervised classification models using the Breast Cancer Wisconsin Diagnostic Dataset from scikit-learn. The objective is to classify tumors as malignant or benign.

Files
Supervised_Classification_Assignment.ipynb - completed Google Colab notebook
Supervised_Classification_Report.pdf - two-page report
README_Supervised_Classification_Assignment.md - project description and run instructions
Models Used

Random Forest Classifier

How to Run
Open Google Colab.
Upload Supervised_Classification_Assignment.ipynb.
Click Runtime > Run all.
Review the outputs, plots, and model results.
Deployment Summary
The best model can be deployed as a REST API using Flask or FastAPI. Monitoring should include data drift, prediction confidence, performance changes, latency, and retraining needs.
